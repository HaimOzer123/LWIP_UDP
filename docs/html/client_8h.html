Detailed Description

Header file for the UDP client implementation.

This file defines the structures, macros, and function prototypes for sending test commands to the server and receiving results.

The client communicates with a UDP server to execute hardware tests on various peripherals, including UART, ADC, Timer, SPI, and I2C. Results are logged to a file for later analysis.

Note
Ensure the client is configured with the correct server IP and port. Use the provided sudo ifconfig command to configure the client's IP address.
