Detailed Description

Header file for UART testing functions.

This file contains definitions and declarations for testing UART communication, including configurations for debugging and handling UART callbacks for error and RX complete events.

Note
Modify the UART handles (huart3, huart5, huart2) to match your hardware configuration as necessary.
