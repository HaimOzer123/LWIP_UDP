Detailed Description

Implementation of UART testing functions.

This file contains the implementation of functions for testing UART communication, including transmission and reception between UART5 and UART2.

Required connections for UART testing:

  • TX5 (PC_12) <--> RX2 (PD_6)
  • RX5 (PD_2) <--> TX2 (PD_5)

The test verifies data integrity over multiple iterations by transmitting a bit pattern and checking the received data for mismatches or errors. It also includes error handling for transmission and reception failures.
