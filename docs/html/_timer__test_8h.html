Detailed Description

Header file for Timer testing functions.

This file provides the declarations for testing timer synchronization between TIM3 and TIM2 on the STM32F756ZG microcontroller.

Note
TIM3 and TIM2 must be connected to appropriate hardware timers in the STM32. Ensure interrupt callbacks are enabled in the HAL configuration.
